# Microserviço de autenticação com Node.js

Neste projeto Temos alguns Endpoints Base que podem ser extendidos da forma mais adequada para seu contexto.

## Usuários

-_GET/user_
-_GET/user/:uuid_
-_POST/users_
-_PUT/users/:uuid_
-_DELETE/users/:uuid_

## Autenticação

-_POST/token_
-_POST/token/validate_
